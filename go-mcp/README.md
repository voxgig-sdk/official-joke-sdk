# official-joke-mcp

MCP server exposing the OfficialJoke SDK as tools, built on the
[official Go MCP SDK](https://github.com/modelcontextprotocol/go-sdk)
and the sibling Go SDK at `../go`.

## Tools

| Tool | Args | Returns |
|------|------|---------|
| `official-joke_list` | `entity`, optional `query` map | First page of records as JSON |
| `official-joke_load` | `entity`, `query` (e.g. `{id:N}`) | Single record as JSON |

JSON schemas for both tools are emitted by the SDK from the `Args`
struct's `json` / `jsonschema` tags — no schema is hand-written.

## Entities

joke | type

## Build

```sh
go build -o official-joke-mcp ./...
```

## Run

```sh
# stdio transport — for spawnable agent installs (default).
./official-joke-mcp

# streamable HTTP transport — share one running server between agents.
./official-joke-mcp -transport http -addr :8080
```

## Install for Claude Code

```sh
claude mcp add --scope user official-joke \
  -- /absolute/path/to/official-joke-mcp -transport stdio
```

After install, restart Claude Code; the `official-joke_list` and
`official-joke_load` tools become available in new sessions.

## Smoke test via HTTP

```sh
./official-joke-mcp -transport http -addr :18080 &

# initialize, grab the session id
curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -D headers \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-06-18","capabilities":{},"clientInfo":{"name":"smoke","version":"0"}}}'

SESSION=$(awk '/Mcp-Session-Id/ {print $2}' headers | tr -d '\r')

curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -H "Mcp-Session-Id: $SESSION" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"official-joke_load","arguments":{"entity":"joke","query":{"id":1}}}}'
```
